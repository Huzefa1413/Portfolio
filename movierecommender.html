NLP - Movie Recommender System

Project information

  • Category: Natural Language Processing
  • Project URL: Click Here

This code builds a movie recommendation system using content-based filtering and integrates it into a Streamlit web interface. It processes movie data, including titles, genres, cast, and crew information, to create movie tags. The code then calculates cosine similarity between movie tags to determine movie recommendations. The recommendations are displayed in the Streamlit interface along with movie posters and links to details. The tool fetches movie details and posters from the TMDB API. The code serializes movie data and similarity scores, making the recommendation process faster. The Streamlit app allows users to input a favorite movie and receive personalized recommendations.
