It enables a student to check all his data such as attendance, registered courses, Timetable, tracking your GPA, Managing dues and balance from a single Application, without the hassle of again and again log in to the student Portal to check his/her information. Our project is a Flask web application that utilizes Selenium to automate web scraping tasks from an educational portal. It offers functionalities for retrieving a user's timetable, ledger, GPA, and attendance information. The goal of the project is to create a user-friendly interface to interact with the educational portal and display the extracted data in a more accessible format.
