Classification - Diabetes Checker

Project information

  • Category: Classification
  • Project URL: Click Here

This code creates a diabetes prediction tool using a Logistic Regression model. It loads diabetes data from a CSV file, preprocesses it by splitting into training and testing sets, standardizing features, and training the classifier. The trained classifier and standard scaler are serialized using the pickle library. A Streamlit web interface is built to input patient's medical information, and upon clicking the "Check Diabetes" button, the model predicts whether the patient has diabetes or not. The tool offers real-time predictions based on patient data, combining data preprocessing, model training, and interactive visualization.
