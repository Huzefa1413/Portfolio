NLP - Chat with PDF

Project information

  • Category: Natural Language Processing
  • Project URL: Click Here

The provided code is a Streamlit app for interacting with multiple PDF documents through a conversational chat interface. Users upload PDFs, and the app extracts text, segments it into chunks, and generates embeddings using HuggingFace's Sentence Transformers. The chat interface employs a retrieval-based conversational model (Flax-Hub) and maintains conversation history. User input triggers responses fetched from the history, displayed alternately as user and bot messages. The app uses Streamlit's interface for user interaction and allows querying documents for information retrieval. It combines PDF text extraction, text chunking, embedding generation, and a conversational chat system.
